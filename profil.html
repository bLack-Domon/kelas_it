Profil Jurusan Rekayasa Perangkat Lunak (RPL) SMK

Profil Jurusan

Rekayasa Perangkat Lunak (RPL) di SMK adalah jurusan yang mempersiapkan siswa untuk menjadi tenaga ahli dalam bidang pengembangan perangkat lunak. Siswa akan mempelajari berbagai keterampilan mulai dari pemrograman dasar hingga pengembangan aplikasi berbasis web dan mobile.

Kurikulum

  • Kelas 10: Pengantar Teknologi Informasi, Dasar-dasar Pemrograman, Logika dan Algoritma
  • Kelas 11: Pemrograman Lanjut, Basis Data, Desain Web
  • Kelas 12: Pemrograman Mobile, Proyek Perangkat Lunak, Magang

Bidang Pekerjaan

  • Programmer
  • Web Developer
  • Mobile App Developer
  • IT Support
  • Network Technician
  • Database Administrator
